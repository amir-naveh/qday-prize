# QDay Prize — Overview

## What Is It?
An open quantum cryptanalysis competition run by **Project Eleven (P11)** challenging participants to break elliptic curve cryptography (ECC) keys using **Shor's algorithm on real quantum hardware**.

## Prize
- **1 Bitcoin** awarded to the team that breaks the largest ECC key

## Deadline
- **April 5, 2026** (rolling submissions accepted until closure)

## Organizer
- **Project Eleven (P11)** — https://projecteleven.com
- Contact: qdayprize@projecteleven.com

## Mission
"Break the largest ECC key possible using Shor's algorithm on a quantum computer."

## Why It Matters
- 256-bit ECC provides security equivalent to a 3072-bit RSA key
- ~2,000 logical (error-corrected) qubits could theoretically break a 256-bit ECC key
- Current quantum hardware has 99–99.9% gate fidelity — still far from full 256-bit, but this competition tracks progress
