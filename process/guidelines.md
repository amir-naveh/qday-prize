# Guidelines

1. **Use Classiq for all quantum code.** Any quantum circuit or algorithm must be implemented using the Classiq platform.
   - Documentation: https://docs.classiq.io/latest/
   - Examples: https://github.com/Classiq/classiq-library

2. **Use only open source files.** The solution may only use publicly available, open source code and data. Do not use any personal or proprietary files, data, or resources.
